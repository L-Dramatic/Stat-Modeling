## 1. 项目概述

### 1.1 项目背景

城市空气质量监测是环境科学和公共健康领域的重要议题。PM2.5作为主要污染物，其浓度受多种气象因子影响，具有复杂的时间依赖性和状态转移特征。本项目旨在构建一个基于**纯统计学方法**（非机器学习）的空气质量监测预警系统，通过多维统计推断、回归建模、分类分析和时间序列预测，实现对PM2.5的归因分析和预警功能。

### 1.2 核心目标

1. **数据探索与预处理**：建立完整的数据清洗和预处理流程
2. **统计推断**：通过假设检验、方差分析、相关分析识别关键因子
3. **回归建模**：构建多种回归模型（OLS、Ridge、Lasso、Bayesian Ridge、GLM）进行归因分析
4. **分类建模**：构建分类模型（Logistic Regression、Naive Bayes、HMM）进行状态识别
5. **时间序列预测**：使用ARIMA模型进行短期预测
6. **系统实现**：开发交互式Web系统，展示所有分析结果

### 1.3 数据来源

- **数据集**: UCI Beijing PM2.5 Data Set
- **数据规模**: 约1.5MB（<10MB，符合课程要求）
- **时间跨度**: 2010-2014年
- **特征**: PM2.5浓度、温度、气压、露点、降雨、风速、风向等

## 4. 改进方案设计

### 4.1 分类模型实现方案

#### 4.1.1 问题转换

将PM2.5连续值转换为分类问题：

```
优良：PM2.5 < 75 μg/m³
轻度污染：75 ≤ PM2.5 < 115 μg/m³
重度污染：PM2.5 ≥ 115 μg/m³
```

#### 4.1.2 模型选择

1. **Logistic Regression（Baseline）**
   - 多分类逻辑回归（Multinomial Logistic Regression）
   - 作为基准模型，用于与HMM对比
   - 优势：简单、可解释性强
   - 劣势：无法利用时间依赖关系

2. **Naive Bayes（贝叶斯分类器）**
   - 高斯朴素贝叶斯（Gaussian Naive Bayes）
   - 体现贝叶斯方法在分类中的应用
   - 优势：概率输出、处理小样本

3. **HMM（高级分类模型）**
   - 利用隐状态转移，捕捉时间依赖
   - 优势：考虑历史状态，准确率更高
   - 用于证明复杂模型的优越性

#### 4.1.3 评估指标

- **准确率（Accuracy）**
- **精确率（Precision）**：每类的精确率
- **召回率（Recall）**：每类的召回率
- **F1-Score**：每类的F1值
- **混淆矩阵（Confusion Matrix）**
- **ROC曲线和AUC值**（二分类或多分类AUC）

### 4.2 贝叶斯方法实现方案

#### 4.2.1 Bayesian Ridge Regression

**目的**：量化参数不确定性，提供后验分布

**实现方式**：
- 使用 `scikit-learn` 的 `BayesianRidge`
- 或使用 `pymc3`/`pymc4` 进行更完整的贝叶斯推断

**输出**：
- 参数后验分布（可视化）
- 可信区间（Credible Intervals）vs 置信区间（Confidence Intervals）
- 不确定性量化说明

#### 4.2.2 Bayesian Classification

**目的**：使用贝叶斯方法进行空气质量分类

**实现方式**：
- Naive Bayes（已包含在分类模型中）
- 或使用贝叶斯逻辑回归

**优势**：
- 提供预测概率分布
- 可以融入先验知识

### 4.3 其他回归模型实现方案

#### 4.3.1 Ridge Regression

**目的**：处理多重共线性问题

**应用场景**：
- 气象因子（温度、气压、露点）存在共线性
- Ridge通过L2正则化稳定系数估计

**对比展示**：
- OLS vs Ridge：系数对比图
- AIC/BIC对比

#### 4.3.2 Lasso Regression

**目的**：特征选择

**应用场景**：
- 从多个气象因子中筛选关键变量
- 通过L1正则化将不重要变量系数压缩为0

**工作流程**：
```
原始特征 → Lasso筛选 → 重要特征列表 → 用于后续建模
```

**可视化**：
- 特征重要性条形图
- 正则化路径图（Regularization Path）

#### 4.3.3 模型对比功能

**对比维度**：
- AIC/BIC值
- R²值
- RMSE/MAE
- 残差分析
- 系数稳定性

**结论**：
- 证明为什么GLM/Bayesian更适合该数据
- 展示模型选择过程

## 8. 评估指标体系

### 8.1 回归模型评估指标

| 指标 | 公式/说明 | 用途 |
|------|---------|------|
| **RMSE** | $\sqrt{\frac{1}{n}\sum(y_i-\hat{y}_i)^2}$ | 衡量预测误差 |
| **MAE** | $\frac{1}{n}\sum|y_i-\hat{y}_i|$ | 平均绝对误差 |
| **R²** | $1-\frac{SS_{res}}{SS_{tot}}$ | 模型解释能力 |
| **AIC** | $2k-2\ln(L)$ | 模型选择（考虑复杂度） |
| **BIC** | $k\ln(n)-2\ln(L)$ | 模型选择（更严格） |
| **Durbin-Watson** | 检验残差自相关性 | 时间序列假设检验 |

### 8.2 分类模型评估指标

| 指标 | 公式/说明 | 用途 |
|------|---------|------|
| **Accuracy** | $\frac{TP+TN}{TP+TN+FP+FN}$ | 整体准确率 |
| **Precision** | $\frac{TP}{TP+FP}$ | 精确率（每类） |
| **Recall** | $\frac{TP}{TP+FN}$ | 召回率（每类） |
| **F1-Score** | $2\cdot\frac{Precision\cdot Recall}{Precision+Recall}$ | F1值（每类） |
| **AUC** | ROC曲线下面积 | 分类性能综合指标 |
| **Confusion Matrix** | 混淆矩阵 | 详细分类结果 |

### 8.3 模型对比维度

1. **性能对比**：RMSE、Accuracy等指标
2. **复杂度对比**：AIC、BIC
3. **可解释性对比**：系数解释、概率输出
4. **适用场景对比**：各模型的优势和局限性
